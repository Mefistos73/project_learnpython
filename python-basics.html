Операторы в Python

В языке программирования Python есть арифметические, логические и операторы сравнения. Ниже приведены примеры основных арифметических операторов:

Операция Пример кода Вывод
Сложение (+)
print(5 + 3)
 8
Вычитание (-)
print(10 - 4)
 6
Умножение (*)
print(7 * 6)
 42
Деление (/)
print(8 / 2)
 4.0
